Question-1: What are the different ways to select an element in the DOM?

1. by ID => getElementById()
2. by Class => getElementsByClassName()
3. by Tag name => getElementsByTagName()
4. by Query selector => querySelector()
5. by Quety selector all => querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML => It insert HTML code from the element.
2. innerText => It insert text from the element.
3. textContent => It insert text from the element (including hidden text).

Question-3: What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন এর মাধ্যমে parent node এর উপরে eventLestener অ্যাড করে ই (event.target) এর মাধ্যমে child দের access করা যায় , এবং event চালানো যায়।

Question-4: What is event bubbling in the DOM?

দেখতে গেলে এটি delegation এর উলটা,
এটিতে child এ click হলে তা bubble হয়ে তার parent এ যায়
এটি চলে html ট্যাগে পউছানো পর্যন্ত , তবে এটিকে থামানো ও যায় .stopPropagration() use করে ।

Question-5: How do you create, add, and remove elements using JavaScript?

1. to create a new element => createElement()
2. to add element into parent => appendChild()
3. to remove element => removeChild()
